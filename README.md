# Vue HTML 2 PDF

Proyecto para convertir componentes de Vue en archivos PDF.

## Requisitos previos

- Node.js

## Instalación

```bash
npm install
```

## Ejecución local

```bash
npm run dev
```

Accede a la aplicación en `http://localhost:5173`

## Tecnologías utilizadas

- Vue 3
- Vite
- vue-to-print
